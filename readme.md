# Academic Homepage

This is my academic homepage hosted on GitHub Pages.

## How to Deploy
1. Fork or clone this repository
2. Rename it to `yourname.github.io`
3. Push to GitHub
4. Enable GitHub Pages (Settings → Pages → main branch)

## Structure
- index.html: Main page
- css/: Stylesheets
- images/: Profile photo
- files/: CV and documents



yourname.github.io/
├── index.html          # 主页面（学术主页）
├── css/
│   └── style.css       # 核心样式（已按截图风格写好）
├── images/
│   └── profile.jpg     # 个人照片
├── files/
│   └── CV.pdf          # 学术简历（可选）
